# 🏎️ SmartCity Rush – Java Pseudo-3D Racing Game

A Java-based **pseudo-3D racing game** built using **LibGDX**, featuring smooth kart controls, AI opponents, collision detection, drifting mechanics, dynamic camera tracking, and persistent leaderboard management using **SQLite**.

---

## 🚀 Features

- Smooth player kart controls
- AI-controlled racing opponents
- Pseudo-3D track rendering
- Dynamic camera tracking
- Collision detection and drifting mechanics
- Background music and sound effects
- SQLite-based leaderboard system
- Modular object-oriented architecture

---

## 🛠️ Technologies Used

- Java
- LibGDX
- JDBC
- SQLite
- Gradle
- Object-Oriented Programming (OOP)

## 🎮 Gameplay

The player controls a racing kart using keyboard controls while competing against AI opponents.

### Controls

| Key | Action |
|-----|--------|
| ↑ | Accelerate |
| ↓ | Brake |
| ← | Turn Left |
| → | Turn Right |
| Z | Drift |

The objective is to finish the race with the highest score while avoiding collisions and maintaining speed through efficient drifting.
